Install Church.IO Profiles

This document assumes you'll be deploying to the free cloud-based service Heroku. If you want to install to another service or your own dedicated server, please hop onto the mailing list and get some help.

1. Create a Facebook "App"

Don't let this part scare you — all you have to do is to log into Facebook and enter a few bits of information to create what Facebook calls an "app."

For the sake of these instructions, let's pretend your church's name is "OneChurch of Grand Rapids."

  1. Go to developers.facebook.com (log into Facebook if you're not already).
  2. Click on "Apps" at the top.
  3. Click the "Create New App" button.
  4. Enter your church name, e.g. "OneChurch of Grand Rapids"
  5. For the namespace, enter a similar name, but only with lowercase characters and dashes, e.g. "onechurch-grandrapids".
  6. On the app settings page, click "Website" and enter URL where your app will be, "http://onechurch-grandrapids.herokuapp.com" (see note below for more help).
  7. Now copy the App ID and App Secret from the top of the page (we'll need this in a bit).
  8. Save changes.

The app URL will be selected a bit later and will be something.herokuapp.com. If the name you choose for your Heroku URL is already taken, you can choose another and come back to Facebook and change the website URL to match.

2. Install Ruby and Git

You'll need to install Ruby and Git on your system (on your own desktop/laptop computer — not your server) in order to download and deploy to Heroku.

  1. Visit rubyinstaller.org (if you're on Windows) or ruby-lang.org (for Mac or Linux) and install the appropriate package.
  2. Once you have Ruby, install the heroku "gem" with the following command:
    gem install heroku
  3. Visit git-scm.com and click on your platform (Windows, Mac, or Linux) on the right to get a download package for Git.

3. Download Profiles

We'll use Git to download the source code for Profiles, and then we'll select the stable version to deploy.

In a command prompt/terminal, change to the directory where you want to download the package, then type the following commands:

git clone git://github.com/churchio/profiles.git
cd profiles
git checkout -t origin/stable

4. Configure Profiles

There are a few settings we need to configure before installation can occur.

  1. Copy config/settings.yml.example to config/settings.yml and edit the file appropriately. Set the fb_app_id and fb_secret to what you copied down from step one above.
  2. Back in the terminal, type:
    rake profiles:heroku:config
  3. Then copy and paste the displayed commands into the terminal.

5. Deploy

git push heroku heroku run rake db:migrate

Visit your new site! The first user to log in is the initial admin user.
